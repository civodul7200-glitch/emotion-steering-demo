Emotion Steering Demo

Injecting emotion directly into a language model's internal state — without instructions.

How does this work?

Normally you'd tell a model "write this in a joyful tone" via a system prompt. Here we do something different: we extract a mathematical emotion direction from the model's internal space and inject it directly into the network's hidden layers as it generates — bypassing the instruction interface entirely.

What you control:

  • Emotion — which direction to inject (joy or anger)
  • Alpha — how strongly to inject it (intensity of the intervention)
  • Prompt — the story seed the model continues

How we measure the effect — two ways:

  • Surface score — a classifier reads the text and detects explicit emotion words
  • Internal alignment — we measure how much the model's hidden state at layer 20 aligned with the emotion vector, even if the text doesn't use obvious emotion words

The gap between these two scores is what's interesting: sometimes the emotion is internally present but not yet expressed on the surface.
